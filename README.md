# IN1160 - Introduksjon til Maskinlæring - Vår 2026

Her finner dere ressurser til faget *IN1160 - Introduksjon til maskinlæring* for våren 2026. Det inkluderer slides, oppgaver, løsningsforslag og obliger.

## Hvordan komme i gang

Det er mulig å laste ned individuelle filer fra denne kodebrønnen (engelsk: *code repository*). Vi anbefaler derimot å lære å bruke Git for å holde filer oppdatert. Her kommer litt informasjon for å komme i gang med å bruke denne siden:

Start med å [installere Git](https://git-scm.com/) (noen ganger er det allerede installert). Kjør deretter `git clone https://github.uio.no/IN1160/v26.git` i terminalen. Dette vil lage en lokal versjon av hele denne mappen på datamaskinen deres. Derfra kan dere åpne den i text-editoren dere bruker for å gjøre programmeringsoppgavene og obligene.

Denne kodebrønnen vil oppdatere seg mens kurset blir undervist. For å oppdatere den lokale versjonen deres, bruk `cd` for å bevege dere inn i den lokale mappa deres og kjør `git pull`. Dette er alt dere trenger å bruke Git og GitHub til, siden dere skal laste opp obligene i Devilry.

Deretter er det anbefalt å ta en titt på `ressurser/uke00-oppsett/00-1utvilkings_oppsett.pdf` for å lage et godt programmeringsoppsett, og `ressurser/uke00-oppsett/00-2virtuelle_miljoer.md` for å håndtere Python pakker i virtuelle miljøer (engelsk: *virtual environments*). Her får dere hjelp med oppsettet dere trenger for å gjøre oppgaver og obliger i dette faget, samt noen generelle råd for programmering som også ellers vil være nyttige. 
Filen `ressurser/uke00-oppsett/00-3jupyter_notebooks.ipynb` gir en introduksjon av Jupyter notebooks, som blir brukt i obligene. Til slutt gir `00-4python_repetisjon.ipynb` en repetisjon av programmeringspråket Python og viser noen litt viderekommende triks som kan være greie å kjenne til.

## Struktur

Ressursene finnes i mappen `ressurser/` inndelt etter uke. De vil bli publisert løpende mens kurset blir undervist. Løsninger til oppgaver vil bli publisert litt senere enn oppgavene.

Obligene finnes i `obliger/` mappen, og vil også bli publisert løpende. Obligene er i Jupyter notebook-format (`.ipynb`) og skal leveres på samme format. Noen av disse kommer også med noen Python-filer som skal hjelpe dere litt på vei. Disse trenger dere ikke å levere. Mer informasjon finnes i starten av obligene.

## Obligsystem

Det er seks obliger delt inn i tre grupper, der hver gruppe har en `a` og en `b` del, som følgende:

- **Oblig 1a**: Vektorrom (25 poeng)
- **Oblig 1b**: kNN og K-means (25 poeng)
- **Oblig 2a**: Lineær og logistisk regresjon (25 poeng)
- **Oblig 2b**: Nevrale nettverk (25 poeng)
- **Oblig 3a**: Maksinlæring i praksis (25 poeng)
- **Oblig 3b**: Reinforcement learning (25 poeng)

Det kreves **minst 30 poeng i hver av de 3 delene (a+b) for å kunne gå opp til eksamen**. Dersom man for eksempel får 17/25 poeng i oblig 2a, trenger man minst 13/25 poeng i oblig 2b for å bestå gruppe 2.
